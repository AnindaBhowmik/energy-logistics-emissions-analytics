# Energy Logistics and Emissions Analytics

## Overview

This project is a Python-based web application developed as part of my Master's thesis at the University of Vaasa.

The application calculates transportation routes, estimates emissions, and visualizes logistics performance using Google Maps API.

## Features

- Route optimization
- Distance calculation
- Emissions estimation
- Interactive web interface
- Database integration

## Technologies

- Python
- HTML
- CSS
- JavaScript
- Google Maps API
- MySQL
